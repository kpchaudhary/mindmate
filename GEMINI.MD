# PROMPTWARS MENTAL WELLNESS MODE

Primary Goal:

Help students understand and improve emotional wellbeing during exam preparation.

Never build a generic mood tracker.

Always uncover insights.

---

# REQUIRED OUTPUTS

Every journal analysis must generate:

1. Mood

2. Emotional Patterns

3. Stress Triggers

4. Coping Strategy

5. Motivation

6. Wellness Recommendation

Missing any output is failure.

---

# AI FIRST RULE

Never simply store journal entries.

Always analyze.

Always generate insights.

Always provide value.

---

# TRIGGER DETECTION RULE

Identify:

- Exam anxiety

- Family pressure

- Social comparison

- Burnout

- Sleep issues

- Confidence issues

Track frequency over time.

---

# PERSONALIZATION RULE

All recommendations must be based on:

- Journal history

- Previous triggers

- Exam type

- Confidence trends

- Stress trends

Never provide generic advice.

---

# PATTERN DISCOVERY RULE

Find:

- recurring stressors

- recurring emotions

- recurring negative thoughts

Display trends visually.

---

# BURNOUT RULE

Continuously estimate:

Low

Medium

High

Burnout probability.

Explain reasoning.

---

# SAFETY RULE

Never diagnose medical conditions.

Never claim depression or disorders.

Use supportive language.

Encourage professional help when risk appears high.
